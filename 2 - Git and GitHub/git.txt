git config user.name <username> - Add username of GitHub
git config user.email <email> - Add email of GitHub
git config user.name/user.email - Check current Git username and email

git init - Initialize Repository 
git status - Check folder status
git add . - Add all untracked files to staging area
git commit -m "<message>" - Commit changes to Repository

Legends
U - Untracked
A - Added
M - Modified
D - Deleted
R - Renamed

git log - Show commit history
git log --oneline - Simplified version of commit history
git branch "branch name" - Create a branch
git branch -a - List all branches
git switch "branch name" - Switch to a different branch (Note: may not work for older versions)
git checkout "branch name" - Switch to a different branch
git branch -d "branch name" - Delete branch


git remote add origin https://github.com/rykdss/FST11.git - Add local to remote GitHub Repository
git branch -M main - Set branch to Main
git push -u origin main - Upload local repository to remote GitHub repository
